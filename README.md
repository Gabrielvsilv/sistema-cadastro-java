# Sistema de Cadastro Java + ViaCEP 🚀

Este é um projeto de backend em Java que realiza o cadastro de pessoas com integração automática de endereço.

## 🛠️ Funcionalidades
- **Busca de CEP:** Integração com a API ViaCEP para preenchimento automático de endereço.
- **Persistência em TXT:** Os dados são salvos localmente em um arquivo de texto.
- **CRUD Completo:** Opções para cadastrar, listar, buscar, editar e remover pessoas.
- **Validação:** Tratamento de erros para entradas inválidas no menu.

## 💻 Tecnologias
- Java 21
- HttpClient (para consumo de API)
- Git/GitHub (Controle de versão)

## 📂 Como rodar
1. Clone o repositório.
2. Importe como um projeto Java no seu IDE (Eclipse/IntelliJ).
3. Execute a classe `Main.java`.
